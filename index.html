Golden Bull Restaurant Loyalty App

Golden Bull Restaurant Loyalty App is a customer rewards platform that allows users to collect stamps on visits and redeem rewards. Restaurant owners can customize promotions, birthday gifts, and sign-up bonuses, while tracking customer engagement and employee performance through detailed analytics.

    Functionality
  • Customer registration and loyalty stamp collection
  • Reward management system for promotions and bonuses
  • Data visualization of customer trends and employee performance
  • Automated email notifications for promotions and birthdays

The application is currently deployed on Azure with a PostgreSQL database and Redis for task queuing.

Reviewee

Reviewee is a web application designed for business owners to showcase their businesses online in order to get user reviews, reservations and also manage their places bookings.

    Functionality
  • User registration and account management
  • Business owners can Create | Edit | Delete restaurants or hotels and view reservations made for them
  • Non-registered users can browse places, view reviews, create and manage reservations
  • Registered users have all non-registered users functionality, plus ability to add places to their favourites and Write | Edit | Delete reviews.

The application is currently deployed on AWS EC2 instance with Nginx as a proxy server and Gunicorn as a local server
